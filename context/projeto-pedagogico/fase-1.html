Desafio 1.2 — Função de validação · arquivo: src/ingestion.py
📥 Entrada: DataFrame pandas 📤 Saída: dict com relatório de qualidade + is_valid 🔗 Regras vêm de: 1.1
🎯 O que construir

Projete validate_dataframe(df) em src/ingestion.py — retorna um dict com três chaves: missing_pct, violations, e is_valid. Os thresholds derivam diretamente do que você mediu no 1.1, não são arbitrários.

  • Ausência: (df.isnull().sum() / len(df) * 100).round(2) filtrado onde maior que zero — entra como missing_pct (dict, não Series)
  • Violações de domínio (4 regras com operador bitwise |): "age_invalid" = (df['age'] < 18) | (df['age'] > 100), "income_negative" = df['MonthlyIncome'] < 0, "debt_ratio_high" = df['DebtRatio'] > 100, "utilization_high" = df['RevolvingUtilizationOfUnsecuredLines'] > 1.5 — cada um com .sum()
  • Flag is_valid: max(missing_pct.values()) < 30 and sum(violations.values()) == 0 — expressão inline; qualquer violação torna o dataset inválido

def validate_dataframe(df):
    null_pct = (df.isnull().sum() / len(df) * 100).round(2)
    missing_pct = null_pct[null_pct > 0].to_dict()
    violations = {}

    # age fora de [18, 100]
    bad = ((df['age'] ) | (df['age'] )).()
    if bad: violations[] = int(bad)

    # MonthlyIncome negativa
    bad = (df['MonthlyIncome'] ).sum()
    if bad: violations[] = int(bad)

    # DebtRatio acima de 100
    bad = (df['DebtRatio'] ).sum()
    if bad: violations[] = int(bad)

    # RevolvingUtilization acima de 1.5
    bad = (df['RevolvingUtilizationOfUnsecuredLines'] ).sum()
    if bad: violations[] = int(bad)

    return {
        'missing_pct': missing_pct,
        'violations': violations,
        'is_valid': max(missing_pct.values(), default=0)  and  == 0
    }

Projete fetch_from_api(n) em src/ingestion.py — simula um payload de bureau de crédito, retorna lista de n dicts com schema fixo. random.seed(42) garante que o mesmo n sempre produza o mesmo dataset.

import json, random, hashlib
from datetime import datetime, timedelta
from faker import Faker
from pathlib import Path

fake = Faker('pt_BR')

def fetch_from_api(n_records):
    random.seed(42)  # mesmo n → mesmo dataset sempre
    base = datetime.now()
    records = []
    for i in range(n_records):
        cpf = fake.cpf().replace('.','').replace('-','')
        records.append({
            "applicant_id": f"APP-{i+1:06d}",
            "cpf_hash": hashlib.sha256(cpf.encode()).hexdigest()[:16],
            "bureau_score": random.randint(300, 900),   # range realista de score
            "active_debts": random.randint(0, 15),
            "requested_at": (base - timedelta(days=random.randint(0, 365))).isoformat()
        })
    Path('data/raw').mkdir(parents=True, exist_ok=True)
    with open('data/raw/bureau_api.json', 'w') as f:
        json.dump(records, f)
    return records
Desafio 1.4 — Unificar fontes e salvar Parquet · arquivo: src/ingestion.py
📥 Entrada: data/raw/cs-training.csv + data/raw/bureau_api.json 📤 Saída: data/staging/credit_applications.parquet 🔗 Alimenta: DuckDB na Fase 2
🎯 O que construir

Projete merge_sources(csv_path, json_path) em src/ingestion.py — consolida o CSV histórico com o JSON de bureau em um Parquet único. A ordem das três etapas é invariante: flags de ausência antes da imputação, senão a informação se perde para sempre.

  • Flags de ausência (primeiro, antes de qualquer .fillna()): income_missing = df['MonthlyIncome'].isnull().astype(int) e dependents_missing = df['NumberOfDependents'].isnull().astype(int) — após imputação, .isnull() retorna tudo False
  • Imputação: MonthlyIncome com .fillna(median()) — não média; o P99 do 1.1 está muito acima do P50, a média seria puxada pelos outliers. NumberOfDependents com .fillna(0) — zero é a leitura semântica correta para ausente aqui
  • Validação pós-merge: confirmar zero nulos em SeriousDlqin2yrs, age, DebtRatio, RevolvingUtilizationOfUnsecuredLines — essas colunas não devem ter nulos em nenhum momento
  • Serialização: df.to_parquet(path, index=False) — index=False evita coluna extra que quebraria o schema da staging SQL na Fase 2

import pandas as pd
from pathlib import Path

def merge_sources(csv_path, json_path):
    df = pd.read_csv(csv_path, index_col=0)
    df['income_missing'] = df['MonthlyIncome'].isnull().astype(int)
    df['dependents_missing'] = df['NumberOfDependents'].isnull().astype(int)
    df['MonthlyIncome'] = df['MonthlyIncome'].fillna(df['MonthlyIncome'].median())
    df['NumberOfDependents'] = df['NumberOfDependents'].fillna(0)
    Path('data/staging').mkdir(parents=True, exist_ok=True)
    df.to_parquet('data/staging/credit_applications.parquet', index=False)
    print(f"Staging: {len(df)} registros → data/staging/credit_applications.parquet")
    return df
